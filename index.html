<fe-text></fe-text>
Simple replacement for input[ type="text" ]
<fe-text placeholder="First Name"></fe-text>
With the placeholder attribute. Placeholder will not minimize on focus.
<fe-text placeholder="Enter Date" data-helper-text="In YYYY-MM-DD format."></fe-text>
With the data-helper-text attribute. It adds hints to what the expected input should be.
<fe-text disabled placeholder="Enter Date" data-helper-text="In YYYY-MM-DD format."></fe-text>
Supports disabled and readonly attributes. A relevant hint will appear instead of the value in data-helper-text. disabled behavior supercedes readonly if both attributes are present.
